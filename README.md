# 🌌 APIVerse - 智能API聚合与可视化平台

> 基于public-apis构建的API聚合平台，集成AI助手帮助开发者快速发现、测试和集成第三方API。

## ✨ 核心功能

- 📚 **API浏览** - 500+免费API，分类浏览、智能搜索
- 🔧 **可视化调试** - 拖拽式API配置，实时预览
- 🤖 **AI助手** - 智能推荐、代码生成、错误诊断
- 📊 **数据可视化** - API响应数据图表展示

## 🛠️ 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React, Next.js, Tailwind CSS |
| 后端 | Node.js, Express, MongoDB |
| AI | OpenAI API |
| 部署 | Vercel, Railway |

## 🚀 快速开始

### 后端
```bash
cd backend
npm install
npm run dev
```

### 前端
```bash
cd frontend
npm install
npm run dev
```
